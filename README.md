# 🎵 TypeScript Music Player

A simple music player built with **TypeScript, HTML and CSS** to practice:
- Type safety & interfaces
- DOM manipulation
- State handling (play / pause / next)
- UI structure and styling

This project is part of my journey learning modern frontend development.

---

## ✨ Features
- Displays song title, artist and album cover
- Uses TypeScript interfaces for strong typing
- Simple playlist handling
- Clean, card-based UI
- Ready for play / pause / next logic

---

## 🛠 Tech stack
- **TypeScript**
- **HTML5**
- **CSS3**
